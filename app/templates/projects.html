StreetEats is a Flask web app that suggests restaurants nearby after automatically accessing your location. Users are also able to add their favorite restaurants to lists after creating an account using our login/register system. This login/register system is built on a PostgreSQL database in Python. The web app is hosted on an EC2 instance and an NginX reverse proxy. StreetEats is containerized using Docker and has its own CI/CD pipeline on Github Actions to allow for automated testing and deployment. Prometheus and Grafana are used for monitoring. StreetEats was one of the Winners of the Batch's Hackathon!

A cross-platform app with React Native that notifies all app users when posts have been made in the app and allows users to view all posts by implementing Firebase Realtime Database in the app. I noticed how it was difficult to keep myself and many other students updated on the university, as keeping up with university information on social media is quite difficult, since it is extremely easy to lose information online on Social Media. Therefore, I thought to solve this problem by creating an app that would send alerts to all students. This feature is implemented with the use of firebase realtime database. The user is prompted with a login system and if the user's info matches the admin's, they will be able to make in app posts. Once the admin makes the post, this information is displayed to all users and is sent to the firebase realtime database.
